Smart Study Buddy

A collaborative mobile platform for student learning resources, note sharing, past papers, and course-based study group discovery.

Smart Study Buddy is a mobile application designed to solve the everyday challenge students face when searching for reliable academic resources and study partners. Built with Flutter and powered by Firebase, the app brings notes, past examination papers, academic information, and peer discussion groups together in one unified platform.

GitHub Repository   Explore Features

About the Project

University and college students routinely struggle to locate organised, course-specific academic resources. Lecture notes and past papers are scattered across personal devices, chat groups, and informal peer networks — making them difficult to find, verify, and share. At the same time, students wanting to revise collaboratively often have no reliable way to discover study groups for the specific courses they are taking.

Smart Study Buddy addresses this by creating a single, student-owned platform where anyone can upload, browse, and download course materials, locate and join discussion groups, and interact with peers — all from their mobile device.

Key Features

📄
Notes & Past Paper Sharing
Upload and download lecture notes and past examination papers organised by course, unit, and academic level. Content is searchable and filterable.

👥
Study Group Discovery
Find and join discussion groups for any course. Students can create new groups, tag them by subject, and coordinate study sessions within the app.

🔔
Notifications
Receive alerts for new uploads, group messages, and activity relevant to your courses so you never miss important updates.

⭐
Ratings & Verification
Rate shared notes and papers to help maintain quality and credibility. Flag irrelevant or incorrect content easily.

🔍
Search & Filter
Quickly find resources and groups by course code, topic, year, or keyword. The discovery system is built for speed and relevance.

👤
Student Profiles
Each student has a profile tracking their contributions, joined groups, and shared resources — encouraging active participation.

Technology Stack

Flutter (Dart) Firebase Authentication Cloud Firestore Firebase Storage Firebase Cloud Messaging GitHub GitHub Pages

The application is built using Flutter for a cross-platform mobile experience (Android and iOS from a single codebase). All backend services are handled by Firebase — Firestore for the database, Firebase Storage for file uploads, Firebase Authentication for user accounts, and Firebase Cloud Messaging for push notifications.

Database Structure

The Firestore database is organised into four top-level collections:

Collection Key Fields Purpose
users uid, name, email, joinedGroups[ ], uploadCount Student profiles and contribution tracking
notes title, courseCode, uploadedBy, fileURL, rating, timestamp Shared lecture notes and past papers
study_groups name, courseCode, description, members[ ], createdBy Course-based discussion and study groups
messages groupId, senderId, text, timestamp In-group chat messages
